This program contains the option to use Command line arguments.
These arguments can be entered in any order and Capitalization does not matter.
They are as follows:

To choose between IPv4 or IPv6 packets:
Enter ipv6 to use IPv6 packets.
If nothing is entered IPv4 packets will be used.

To use Sequential Acknowledgement or TCP like sliding windows:
Enter tcp for sliding window simulation.
If nothing is entered Sequential acks will be used

To receive all packets or simulate dropping 1% of the packets sent:
Enter "drop" to simulate dropping 1% of packets sent from the server.
If nothing is entered no packets will be dropped.
